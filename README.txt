L’ASCENSION DE L’HÉRÉTIQUE

Application web dark fantasy, utilisable sur téléphone et ordinateur.

FONCTIONNALITÉS
- Quêtes principales et secondaires
- XP, niveaux, rangs, or et discipline
- Boss avec points de vie
- Boutique de récompenses
- Score quotidien strict
- Journal d’actions
- Export/import de sauvegarde
- Fonctionne hors connexion après installation

LANCEMENT SIMPLE SUR ORDINATEUR
1. Décompressez le dossier.
2. Ouvrez un terminal dans ce dossier.
3. Lancez :
   python -m http.server 8000
4. Ouvrez http://localhost:8000 dans un navigateur.

SUR ANDROID
Pour l’installer comme une application, le dossier doit être hébergé sur un petit serveur web HTTPS.
Une solution simple est d’utiliser GitHub Pages, Netlify ou Cloudflare Pages.
Une fois le site ouvert dans Chrome, utilisez “Ajouter à l’écran d’accueil” ou le bouton “Installer”.

IMPORTANT
Les données sont stockées localement dans le navigateur. Utilisez régulièrement “Exporter la sauvegarde”.
